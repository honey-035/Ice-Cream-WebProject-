# 🍦 NOTO - Responsive Ice Cream Website

A modern and responsive frontend website for **NOTO Ice Cream**, built using **HTML5, CSS3, and JavaScript**. The project showcases a clean, visually appealing interface with smooth navigation, engaging sections, and a mobile-friendly design to provide an enjoyable browsing experience.

---

## 🌐 Live Demo

🔗 **Website:**  
https://honey-035.github.io/Ice-Cream-WebProject-/

---

## 📖 About the Project

NOTO is a responsive frontend web application inspired by a premium ice cream brand. The website features an attractive landing page, product highlights, about section, and contact page while maintaining a clean and user-friendly interface.

The primary objective of this project was to strengthen frontend development skills by building a complete responsive website from scratch and deploying it using GitHub Pages.

## 🚀 Getting Started

```bash

```

### Navigate to the project folder

```bash
cd Ice-Cream-WebProject-
```

### Run the project

Simply open **index.html** in your browser.
